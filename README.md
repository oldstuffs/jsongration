## Setup

<details>
<summary>Gradle</summary>

```gradle
repositories {
    mavenCentral()
}

dependencies {
    implementation("io.github.portlek:jsongration:${version}")
}
```
</details>

<details>
<summary>Maven</summary>

```xml
<dependencies>
    <dependency>
      <groupId>io.github.portlek</groupId>
      <artifactId>jsongration</artifactId>
      <version>${version}</version>
    </dependency>
</dependencies>
```
</details>
